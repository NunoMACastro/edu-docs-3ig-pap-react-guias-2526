# Materiais Didáticos - React + Node + MongoDB (12.º Ano)

Repositório com materiais em Markdown para frontend (React), backend (Node.js) e base de dados (MongoDB Atlas).

## Como usar

-   Escolhe o módulo que vais estudar e começa pelo README desse módulo.
-   Faz os exemplos no teu projeto e resolve os exercícios.
-   Os blocos [EXTRA] são opcionais para quem quer aprofundar.

## Pré-requisitos

-   **Node.js + npm:** idealmente Node 20 LTS (Node 18+ funciona).
-   **Editor de código:** VS Code ou equivalente.
-   **Terminal:** para correr comandos (`npm`, `node`).
-   **Browser atualizado:** Chrome, Edge ou Firefox.
-   **Conta MongoDB Atlas:** gratuita (Free Tier).
