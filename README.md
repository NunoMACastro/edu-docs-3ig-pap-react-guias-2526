# Materiais Didáticos - React + Node (12.º Ano)

Repositório com materiais em Markdown para frontend (React) e backend (Node.js).

## Como usar

-   Escolhe o módulo que vais estudar e começa pelo README desse módulo.
-   Faz os exemplos no teu projeto e resolve os exercícios.
-   Os blocos [EXTRA] são opcionais para quem quer aprofundar.

## Pré-requisitos

-   **Node.js + npm:** idealmente Node 18+.
-   **Editor de código:** VS Code ou equivalente.
-   **Terminal:** para correr comandos (`npm`, `node`).
-   **Browser atualizado:** Chrome, Edge ou Firefox.

## Índice rápido

-   **React (frontend):** [React/README.md](React/README.md)
-   **Node (backend):** [Node/README.md](Node/README.md)

## Portas típicas

-   **React (Vite):** `http://localhost:5173`
-   **Node (API):** `http://localhost:3000`

## Troubleshooting rápido

-   Para problemas de Node, consulta [Node/15_troubleshooting.md](Node/15_troubleshooting.md).
-   Para problemas de React, confirma o `div#root` e a consola do browser.
