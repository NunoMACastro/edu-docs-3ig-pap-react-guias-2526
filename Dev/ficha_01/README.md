# React - Primeiros Passos (frontend)

Projeto simples em React que cobre os temas dos 5 primeiros ficheiros da pasta React.
Cada secao mostra um exemplo curto e direto para estudar e modificar.

## Como correr

1. npm install
2. npm run dev
3. Abre o link indicado pelo Vite

## Estrutura

- index.html: contem o div com id root
- src/main.jsx: cria a root do React
- src/documents: componentes da pagina
- src/styles.css: estilos base

## Componentes principais

- src/App.jsx: pagina principal e sequencia de passos
- src/documents/Fundamentos.jsx: conceitos base e setup
- src/documents/JsxComponentes.jsx: JSX e expressao
- src/documents/PropsComposicao.jsx: props e children
- src/documents/EstadoEventos.jsx: useState e eventos
- src/documents/ListasCondicionais.jsx: listas, key e condicionais
